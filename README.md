# Algorithm Analysis

This repository contains course materials, implementations, notes, assignments, and practice problems for the **Algorithm Analysis** course.

## 📚 Course Overview

The course focuses on the design, analysis, and optimization of algorithms. Topics include algorithm correctness, complexity analysis, graph algorithms, dynamic programming, greedy techniques, computational geometry, linear programming, and NP-completeness.

---

## 📖 Course Contents

### 1. Introduction to Algorithms
- What is an Algorithm?
- Algorithm Design
- Algorithm Analysis
- Correctness of Algorithms

### 2. Greedy Algorithms
- Introduction to Greedy Strategy
- Greedy Choice Property
- Fractional Knapsack Problem
- Activity Selection Problem
- Huffman Encoding
- Task Scheduling Problem
- Coin Change Problem
- Minimum Spanning Trees
  - Kruskal's Algorithm
  - Prim's Algorithm

### 3. Divide and Conquer
- Introduction to Divide and Conquer
- Design Techniques
- Quick Sort
- Merge Sort
- Proof of Correctness
- Runtime Analysis

### 4. Dynamic Programming
- Introduction to Dynamic Programming
- Principle of Optimality
- Optimal Substructure Property
- Assembly Line Scheduling
- Matrix Chain Multiplication
- Longest Common Subsequence (LCS)
- Viterbi Algorithm
- Travelling Salesperson Problem (TSP)

### 5. Graph Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Flow Networks
- Single Source Shortest Path
- All Pairs Shortest Path

### 6. Linear Programming
- Overview of Linear Programming
- Problem Formulation
- Simplex Algorithm
- Integer Linear Programming

### 7. Advanced Topics
- Computational Geometry
- Number Theoretic Algorithms
- String Matching Algorithms

### 8. NP-Completeness and Approximation Algorithms
- NP Completeness
- Polynomial Time Verification
- Reducibility
- NP Complete Problems
- Approximation Algorithms

## 🎯 Learning Objectives

After completing this course, I should be able to:

- Analyze algorithm efficiency using asymptotic notation.
- Design algorithms using various paradigms.
- Prove algorithm correctness.
- Solve optimization problems using Greedy and Dynamic Programming.
- Apply graph algorithms to real-world problems.
- Understand computational complexity classes.
- Identify NP-Complete problems and approximation techniques.
